<h1 align="center">Era.js Core</h1>
<div align="center">最新版本：v0.1.0-alpha+dev</div>

## 概述

>   更新于：191111

Era.js Core 是 Era.js 引擎的最简形式，里面维护着引擎的核心代码。

文件夹主要有两个：

-   FrontEnd
    -   前端代码
    -   编程语言：TypeScript
-   BackEnd
    -   后端代码
    -   编程语言：Python3
